To Do Application

About

This application is used to create a simple to do list for every date tasks.

  • You can add tasks to the list by simply entering a name of the task plus the due date and click the add button.
  • If you want to delete a task click the delete button to the corresponding item.
  • If you want to change the name of the task, enter the new name and click the update button.
  • If you want to change the due date, enter the new date and click the update date button.
  • The number of to do items is listed at the bottom of the list.
